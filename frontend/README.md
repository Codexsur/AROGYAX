## Overview

The frontend module of **AROGYA-X** provides the user interface for interacting with the healthcare assistant. It enables users to input symptoms, view health recommendations, access emergency services, and interact with the AI chatbot in a simple and user-friendly environment.

The frontend focuses on accessibility, responsiveness, and real-time interaction.

---

## Responsibilities

- Display chatbot interface
- Capture user health queries
- Provide symptom input forms
- Show emergency alerts
- Display nearby hospital information
- Enable multilingual interaction
- Support voice-based input
