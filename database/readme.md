# Database — AROGYA-X

## Overview

The database module manages all persistent data used by the AROGYA-X system. It stores structured healthcare information and user interaction records.

---

## Data Stored

- User profiles
- Symptom history
- Conversation logs
- Emergency alerts
- Appointment records
- System analytics

---

## Supported Databases

- MongoDB
- PostgreSQL
- Supabase
- Firebase (optional)

---

## Structure

- User collection
- Medical knowledge base
- Emergency records
- Reminder schedules

---

## Security

- Sensitive data protected via environment variables
- No credentials stored in repository
- Access controlled through backend services

---

## Purpose

Ensures reliable data storage and fast retrieval for AI-based healthcare analysis.
